# 1-1. 과정 소개

## 1. 클린 코드에 대해 고민하기

- 타인이 정의한 답을 의심합니다.
- 배움에 열린 태도를 가집니다.
- 직접 생각하고 또 고민합니다.
- 클린 코드가 무엇인지를 자바스크립트를 통해 학습합니다.
  - clean code for JS
  - JS for clean code
  - clean code < JS
- 흔히 알려진 자바스크립트 코드 스타일에 대한 견해를 탐구합니다.

## 2. JavaScript 특성 파악하기

- 몽키 패치?

  - 타입이 동적으로 형변환되는 자연스러운 변화
  - 런타임 중인 프로그램의 내용이 변경되는 행동

- 안티 패턴?
  - 몽키 패치라고도 부른다?
  - 소프트웨어 공학 분야 용어이며, 실제 많이 사용되는 패턴이지만 비효율적이거나 비생산적인 패턴을 의미
